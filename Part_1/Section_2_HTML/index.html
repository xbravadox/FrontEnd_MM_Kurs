Formatowanie textu

Pogrubiony tekst w tagu 'b'. - wartość CSS
Pogrubiony tekst w tagu 'strong'. - wartość semantycza

Teks pisany kursywą w tagu 'i' - wartość CSS
Teks pisany kursywą w tagu 'em' - wartość semantycza

Podkreślenie tekstu w tagu 'u'. - wartość CSS

Przekreślony tekst w tagu 'del'. - teoretycznie usunięty, ale jednak widoczny.
Przekreślony tekst w tagu 's'. - zawartość nieaktualna.

Wyszczególnienie fragmentu tekstu kolorem w tagu 'mark'.


2 2 =4 - indeks górny w tagu 'sup'.

2 n >4 - indeks dolny w tagu 'sub'.
